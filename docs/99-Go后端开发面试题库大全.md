## 1. Go 语言基础与进阶

### 🟢 基础

- [ ] **Go vs Java**：为什么选择 Go？相比 Java 有什么优势？
- [ ] **Slice**：Slice 和 Array 的区别？底层结构是什么？扩容机制是怎样的？
- [ ] **Map**：Map 是线程安全的吗？底层结构 (`hmap`) 是什么？遍历顺序是固定的吗？
- [ ] **Defer**：多个 defer 的执行顺序？defer 修改具名返回值会发生什么？
- [ ] **Interface**：`nil` 接口的坑（`var i interface{} = (*int)(nil)` 是 nil 吗？）

### 🟡 进阶

- [ ] **内存逃逸**：什么是内存逃逸？哪些场景会触发逃逸？（栈 -> 堆）
- [ ] **GC (垃圾回收)**：Go 的三色标记法是什么？写屏障 (Write Barrier) 解决了什么问题？
- [ ] **Context**：Context 的作用？`Value`、`Cancel`、`Timeout` 怎么用？
- [ ] **反射**：`reflect` 包怎么用？为什么尽量少用反射？

### 🔴 困难/实战

- [ ] **PProf 排查**：线上 CPU 100% 怎么排查？内存泄漏怎么查？
- [ ] **Panic 捕获**：子协程 panic 会导致主进程挂掉吗？如何优雅恢复？

## 2. 并发编程 (GMP & 锁)

### 🟢 基础

- [ ] **Goroutine**：协程和线程的区别？为什么协程轻量？
- [ ] **Channel**：无缓冲 vs 有缓冲 Channel 的区别？已关闭 Channel 读写会怎么样？
- [ ] **Mutex**：互斥锁 (`sync.Mutex`) 和读写锁 (`sync.RWMutex`) 的区别？

### 🟡 进阶

- [ ] **GMP 模型**：详细解释 G、M、P 的作用和调度流程。
- [ ] **Work Stealing**：P 是怎么偷其他 P 的任务的？为什么要从尾部偷？
- [ ] **Sync.Map**：为什么比 `Mutex + Map` 快？（读写分离、空间换时间）
- [ ] **WaitGroup & ErrGroup**：如何等待一组协程结束？发生错误怎么处理？

### 🔴 困难/实战

- [ ] **调度细节**：M 什么时候会休眠？调度器怎么防止全局队列饿死？
- [ ] **并发模式**：手写生产者-消费者模型？手写并发安全队列？
- [ ] **锁底层**：Mutex 是悲观锁还是乐观锁？自旋锁 (Spin Lock) 是什么？

## 3. 数据库 (MySQL)

### 🟢 基础

- [ ] **索引**：B+ 树和 B 树的区别？为什么 MySQL 用 B+ 树？
- [ ] **事务**：ACID 是什么？
- [ ] **隔离级别**：读未提交、读已提交、可重复读、串行化分别解决了什么问题？

### 🟡 进阶

- [ ] **MVCC**：多版本并发控制原理？Read View 是怎么工作的？
- [ ] **锁**：行锁、表锁、间隙锁 (Gap Lock)、临键锁 (Next-Key Lock)。
- [ ] **幻读**：MySQL 怎么解决幻读的？（快照读 vs 当前读）
- [ ] **日志**：Binlog、Redo Log、Undo Log 的区别和作用？

### 🔴 困难/实战

- [ ] **慢 SQL 优化**：Explain 怎么看？Using filesort / Using temporary 如何优化？
- [ ] **深分页**：`Limit 1000000, 10` 怎么优化？(子查询 ID / 游标)
- [ ] **主从延迟**：读写分离导致读不到最新数据怎么办？

## 4. 缓存 (Redis)

### 🟢 基础

- [ ] **数据类型**：常用 5 种类型及其使用场景？(String, List, Hash, Set, ZSet)
- [ ] **持久化**：RDB 和 AOF 的优缺点？

### 🟡 进阶

- [ ] **过期策略**：Redis 是怎么删除过期 Key 的？（定期 + 惰性）
- [ ] **缓存异常**：
  - **缓存穿透**：查不存在的数据（布隆过滤器）
  - **缓存击穿**：热点 Key 过期（互斥锁）
  - **缓存雪崩**：大量 Key 同时过期（随机时间）
- [ ] **ZSet 底层**：跳表 (SkipList) 原理？时间复杂度？

### 🔴 困难/实战

- [ ] **双写一致性**：更新 DB 后删缓存？延迟双删？Canal 订阅？
- [ ] **分布式锁**：`setnx` 怎么实现？看门狗机制 (Redisson) 是什么？
- [ ] **大 Key/热 Key**：怎么发现？怎么处理？
